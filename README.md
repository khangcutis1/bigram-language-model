## 🚀 Getting Started

This guide helps you learn how language models work by building a Bigram model from scratch using pure Python and PyTorch. This tutorial is for beginners, and every line is clearly commented.

## 🌐 What is a Bigram Language Model?

A bigram language model predicts the next word in a sentence based on the previous word. This simple approach is a great introduction to the world of language processing. By understanding bigrams, you will grasp the basics of how language models function.
